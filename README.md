# d8

## 运行步骤
### 安装v8引擎环境
因为codeforce上运行JS是利用v8引擎，我们必须下载v8的二进制包。对于window系统，官网给我们提供安装包。
[下载链接](http://pan.baidu.com/s/1sk9juqT)
### 配置环境变量
比如我们下载解压到D:\v8文件夹，把这个路径加到path环境变量中就ok了
### 在cmd中进入该工程目录，输入 d8 main.js 来编译运行代码，如下图：
